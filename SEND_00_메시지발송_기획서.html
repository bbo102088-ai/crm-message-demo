1 발신번호 선택 (Segment Control)
1-1. Default

기본 발신번호가 선택된 상태로 화면 진입.

1-2. 목록 선택
  • 사무국에서 미리 등록된 번호 목록에서 선택
  • 번호는 환경설정 페이지에서 관리
  • 등록된 번호가 있어야 발신번호 Confirm 됨
2 채널 선택 (탭)
문자
카카오톡
  • 문자 / 카카오톡 탭 전환
  • 문자 탭 → 카카오로 이동 시: 작성된 내용 초기화
  • 카카오 탭 → 문자로 이동 시: 마찬가지로 작성 내용 초기화
3 발송 대상 선택
2-1. 조건 선택
  • 드롭다운 선택 시 조건 선택 기능 활성화
  • 활성화되면 발송 버튼 영역에 [발송하기] 버튼 노출
⚠️ 수기입력 방식인지 확인 필요 (기획서 재확인 요청)
2-2. 직접 입력 (수기 선택)

환자를 직접 검색하여 등록하는 기능

2-3. 소모로 선택 (Dropdown Control)
2-5. 저장한 조건 선택

이전에 저장된 조건 목록에서 불러오는 기능.

  • Default 상태: 없는 상태 / 선택된 상태 / 리스트 펼쳐진 상태
  • 삭제: 데이터 있는 상태에서 삭제 가능
2-5a. [저장한 조건 선택] 클릭 시 리스트 노출
  • Default: 리스트 없는 상태에서의 UX 표시
  • 리스트 선택 시 Filter 상태 표시 filter 상태값 확인 필요
  • 리스트 선택 취소 시 기존 상태로 복귀
2-5b. 저장한 조건 리스트 더블 클릭 시 Confirm

저장한 조건을 더블 클릭 시 조건 상세 Form 노출

4 메시지 작성
  • 광고성 체크 옵션
  • 이미지 추가: JPG, PNG 파일 (최대 3개, 각 5MB 이하)
  • AI 작성 기능
  • SMS / LMS 자동 전환 (byte 초과 시)
광고성 툴팁 — 이미지 첨부 수에 따른 레이아웃
첨부 수표시 방식
1개단일 이미지
2개2열 이미지
3개3열 이미지
5 발송 시간 선택

[즉시 발송] 또는 [예약 발송] 버튼으로 선택.

3-1. 즉시 발송 / 예약 발송 탭
즉시 발송
예약 발송
예약 발송 선택 시
3-2. 예약 날짜 선택 (Selected A)
  • 예약 발송 버튼 클릭 후 달력 표시
  • 최소 3일 이후 날짜만 선택 가능
3-2b. Combo-Box 외부 클릭 시

[확인] 버튼 없이 달력을 닫을 때의 동작

3-3. 예약 시간 선택
  • 달력에서 날짜 선택 후 팝업 Form으로 시간 선택
  • 달력에서 예약 날짜와 발송 아이콘을 선택하면 발송 시간도 같이 선택됨
  • 오전 / 오후 선택 → 시:분 입력 (예: 오전 10:00)
3-4. 예약 시간 확정

예약 시간 설정 완료 상태. [취소] / [확인] 버튼 표시

3-5. 예약 날짜 + 시간 모두 선택 완료

날짜 및 시간이 함께 표시됨 (예: 2025-09-15 (목) 오전 10:00)

7 발신번호
5-1a. 등록된 발신번호가 없을 시 Confirm (프로그램 실행 시)
  • [CRM 클릭] → 환경설정으로 이동
  • [넘어가기] → 발신번호 없이 프로그램 실행 계속
5-2. 발신번호 선택

등록된 발신번호 목록에서 선택

5-8. 발신번호 목록 (기본 설정 있을 시)
5-9. 예약 시간 기준 달력
8 임시 저장

저장 범위: 채널, 보낼 대상, 보낼 시간 선택, 메시지 입력, 광고 여부, 이미지 추가 등 모든 데이터 포함

6-1. 자동 임시 저장
  • 조건: 메시지 내용 1자 이상 AND 불러온 대상 1건 이상 존재 상태에서 페이지 이탈 시 자동 저장
  • 저장 시 토스트 알림 노출
  • 보관 기한: 영구 보관하지 않으며, 저장 시점 기준 7일 동안 유지됨
6-2. 임시저장 불러오기 Confirm

사용자가 메시지 작성 페이지 재진입 시, 임시 저장 데이터가 존재하면 Confirm 모달 노출.

💬 Confirm 문구: "임시 저장된 메시지가 있습니다. 불러오시겠습니까?"
